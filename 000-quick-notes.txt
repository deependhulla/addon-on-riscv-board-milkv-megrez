Downlaod SD Card Image of RockOS (Debian 13) for Milkv Megrez Board

https://fast-mirror.isrc.ac.cn/rockos/images/generic/20250630_20250818/sdcard-rockos-20250818-234921.img.zst

Uncompress and burn it to a blank SATA SSD to boot with SATA SSD on Milkv Megrez 
it works with MicroSSD too.
I used 128GB SSD

 

Install git and vim package to download this repo and work with vim


apt-get -y install vim git
cd /opt/
git clone https://github.com/deependhulla/addon-on-riscv-board-milkv-megrez
